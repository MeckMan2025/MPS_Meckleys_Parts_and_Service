# Meckley's Parts & Service Website

This is the official website for **Meckley's Parts & Service**, a family-owned auto repair, towing, and salvage business in Melbourne, Iowa.

## Project Overview

The website is a static HTML/CSS/JS site designed to be fast, responsive, and easy to maintain. It features a custom "Rural Grit" design system that reflects the company's brand identity.

### Key Features
- **Responsive Design:** Optimized for mobile devices and desktops.
- **Service Pages:** Detailed information on Auto Repair, Towing, Scrap, and Welding.
- **Squareville:** A dedicated section for 1973-1987 Chevy Squarebody enthusiasts.
- **Parts Request:** Call-to-action focused design to drive phone inquiries.
- **Contact Info:** Integrated Google Maps and clear hours of operation.

## Tech Stack

- **HTML5:** Semantic structure.
- **CSS3:** Custom properties (variables) for the design system.
- **JavaScript:** Minimal vanilla JS for mobile navigation.
- **Fonts:** Google Fonts (Oswald & Inter).
- **Icons:** FontAwesome (CDN).

## Local Development

1. Clone the repository.
2. Open the project folder.
3. Serve the files using a local server (e.g., VS Code Live Server or Python).

```bash
# Python 3 example
python3 -m http.server 8080
```

4. Open `http://localhost:8080` in your browser.

## Deployment

This site is static and can be deployed to any static hosting provider such as:
- Netlify (Recommended)
- Vercel
- GitHub Pages
- AWS S3 / CloudFront
